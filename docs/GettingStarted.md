# Getting Started

Welcome to AI Design Operating System (AI-DOS).

This guide will help you get started with the project.

---

# What is AI-DOS?

AI-DOS is an open-source Design Operating System for AI Agents.

It provides a structured design knowledge base that helps AI generate consistent, high-quality digital products.

---

# Who is it for?

AI-DOS is designed for:

- Product Designers
- Frontend Developers
- AI Engineers
- Indie Hackers
- Startup Teams
- Anyone building products with AI

# How AI-DOS Works

The workflow is simple:

1. Load the required knowledge.
2. Apply the design principles.
3. Select the appropriate pattern.
4. Build with reusable components.
5. Review the final result.

---

# Recommended Workflow

1. Read the documentation in `docs/`.
2. Explore the knowledge modules.
3. Load the appropriate Skill.
4. Use the provided patterns.
5. Follow the review checklist before publishing.
