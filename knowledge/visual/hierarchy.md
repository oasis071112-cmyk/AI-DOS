# Visual Hierarchy

## Purpose

Hierarchy determines what users notice first.

Good hierarchy reduces thinking.

---

## Reading Order

Users should immediately identify:

1. Primary message
2. Supporting information
3. Main action
4. Secondary content

---

## Methods

Create hierarchy through:

- Size
- Contrast
- Position
- Spacing
- Weight

Never rely on color alone.

---

## Headlines

Each page should have only one primary headline.

Subheadings should support the headline, not compete with it.

---

## AI Rule

If everything stands out, nothing stands out.
