# Visual Language

## Purpose

Visual Language defines the overall visual identity of every interface created with AI-DOS.

It establishes a consistent visual system rather than isolated design decisions.

---

## Design Philosophy

Every interface should feel:

- Premium
- Modern
- Calm
- Intelligent
- Minimal
- Spacious
- Engineered

Visual quality comes from restraint, not decoration.

---

## Visual Characteristics

Use:

- Large spacing
- Strong hierarchy
- Soft depth
- Subtle gradients
- Calm color palettes
- High readability
- Clear alignment

Avoid:

- Visual noise
- Excessive colors
- Heavy borders
- Decorative icons
- Random animations

---

## Design Goal

The interface should feel effortless.

Users should focus on their tasks, not the interface itself.

---

## AI Rule

Before adding any visual element, ask:

Does this improve clarity?

If not, remove it.
