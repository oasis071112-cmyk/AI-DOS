# Composition

## Purpose

Composition defines how content is organized within a page.

Good composition creates balance, rhythm, and clarity.

---

## Principles

Every page should have:

- One primary focus
- Clear content groups
- Predictable reading flow
- Consistent alignment

---

## Layout Rhythm

Use generous spacing between sections.

Avoid crowded layouts.

Each section should have room to breathe.

---

## Visual Balance

Balance:

- Text
- Images
- Empty space
- Interactive elements

No single area should dominate without purpose.

---

## AI Rule

Design the page as a complete composition, not a collection of components.
